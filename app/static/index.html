Active Slot 不會在下載、patch 或驗證期間被修改。

          
更新只會手動觸發。系統在另一個 Slot 完成 checkout、managed patches、Python 編譯與 import smoke test後才切換；失敗時現行 Slot 保持不變。若新版的 pyproject.toml 或 uv.lock 有變更，更新後必須重新執行 docker compose up --build -d。
